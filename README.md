# SpaceX Launch Analysis Capstone (IBM/Coursera)

This repository contains notebooks and code artifacts for the capstone:
- Data collection (SpaceX API + simple web scraping)
- Data wrangling
- EDA (visual + SQL)
- Interactive analytics (Folium, Plotly Dash)
- Predictive classification (baseline)

## Quick Start
1. Create a Python 3.9+ virtual environment.
2. `pip install -r requirements.txt`
3. Run notebooks in order from `/notebooks`.
4. (Optional) Launch the Dash app: `python app/dash_app.py`
5. Export presentation PPTX to PDF for submission.

## Structure
```
/data              # intermediate CSVs (created by notebooks)
/notebooks         # Jupyter notebooks (placeholders provided)
/sql               # optional: .sql files for queries
/app               # Plotly Dash app
/maps              # store Folium map HTML/snapshots
```
